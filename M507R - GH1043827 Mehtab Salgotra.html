1. Problem Statement



A loss of tuition in higher learning institutions would lead to student dropout and also damage completion-rate surveys as a model of national funding. Besides the institution, early leave reduces life time earnings of the student and has been known to have psychological impacts (Tinto, 1987).

Dataset: The dataset by Realinho et al. (2022), called Predict Students Dropout and Academic Success, has 4,424 records of students that studied in a Portuguese polytechnic and contains 36 features, including demographics, admission grades, semester performance, and macroeconomic indicators.

Business problem: Find students who are at risk during the first year of academic work in order to intervene with the advisors before it is too late to be concerned with student engagement.

Formulation ML: Three-classification: Dropout, Enrolled, Graduate with a deep MLP trained in Keras/TensorFlow on tabular data.

Data collection in production: The features would be mined in the university student information system upon semester end thus causing an immediate model inference call which automatically updates the risk score of each student.

Value: The fact that at least 5 percent of students who might otherwise drop out would stay with the institution saves it significant tuition income and enhances the institutional completion statistics.

5. Model Architecture and Training




I chose a four layer MLP (256-128-64-32) with Batch Normalisation, Dropout (0.3), L2 regularisation, Adam optimiser, EarlyStopping (patience=15), and ReduceLROnPlateau - these parameters were selected based on the experiments in Section 6.

# Build the best MLP model identified through experiments
def build_best_model(input_dim, num_classes=3):
    model = models.Sequential([
        # Hidden layer 1
        layers.Dense(256, kernel_regularizer=regularizers.l2(0.001), input_shape=(input_dim,)),
        layers.BatchNormalization(),
        layers.Activation("relu"),
        layers.Dropout(0.3),

        # Hidden layer 2
        layers.Dense(128, kernel_regularizer=regularizers.l2(0.001)),
        layers.BatchNormalization(),
        layers.Activation("relu"),
        layers.Dropout(0.3),

        # Hidden layer 3
        layers.Dense(64, kernel_regularizer=regularizers.l2(0.001)),
        layers.BatchNormalization(),
        layers.Activation("relu"),
        layers.Dropout(0.3),

        # Hidden layer 4
        layers.Dense(32, kernel_regularizer=regularizers.l2(0.001)),
        layers.BatchNormalization(),
        layers.Activation("relu"),
        layers.Dropout(0.3),

        # Output layer
        layers.Dense(num_classes, activation="softmax")
    ])

    model.compile(
        optimizer=optimizers.Adam(learning_rate=0.001),
        loss="categorical_crossentropy",
        metrics=["accuracy"]
    )
    return model

# Figure 5: Permutation feature importance on test set — computed manually
# sklearn's permutation_importance requires a fit method which Keras does not have
# so we implement the permutation loop directly

from sklearn.metrics import f1_score

def compute_permutation_importance(model, X, y_true, n_repeats=10, random_state=42):
    rng = np.random.RandomState(random_state)

    # Baseline macro F1 on unshuffled test set
    y_pred_base = np.argmax(model.predict(X, verbose=0), axis=1)
    baseline_f1 = f1_score(y_true, y_pred_base, average="macro")

    n_features = X.shape[1]
    importances = np.zeros((n_features, n_repeats))

    for feat_idx in range(n_features):
        for repeat in range(n_repeats):
            X_permuted = X.copy()
            # Shuffle only the current feature column
            X_permuted[:, feat_idx] = rng.permutation(X_permuted[:, feat_idx])
            y_pred_permuted = np.argmax(model.predict(X_permuted, verbose=0), axis=1)
            permuted_f1 = f1_score(y_true, y_pred_permuted, average="macro")
            # Importance = drop in F1 when feature is shuffled
            importances[feat_idx, repeat] = baseline_f1 - permuted_f1

    return importances

print("Computing permutation importance — this may take a few minutes...")

    shap_dropout = shap_array

    shap_dropout = shap_array[0]
